```

(2) JVM에서 바이트코드를 실행하면서 .Class 파일을 클래스 로더 컴포넌트를 활용해 메모리에 로드
(3) Java 실행 엔진이 바이트코드를 기계어로 변환하고 해당 코드가 플랫폼에서 실행
(4) 실행결과를 멀티스레드 영역에 따라 런타임 데이터 영역에 저장
(5) Gabage Collecter를 통해 메모리를 자동으로 관리
```
